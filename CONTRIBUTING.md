# 기여 가이드 (Contributing Guide)

Wine Tracker 프로젝트에 관심을 가져주셔서 감사합니다! 여러분의 기여가 이 프로젝트를 더 좋게 만듭니다.

## 기여 방법

### 버그 신고 (Bug Reports)
버그를 발견하셨다면, Issue 탭에서 'Bug Report' 템플릿을 사용하여 신고해 주세요. 다음 정보를 포함해 주시면 큰 도움이 됩니다:
- 발생한 문제에 대한 명확한 설명
- 문제를 재현하는 단계
- 예상되는 동작과 실제 동작
- 스크린샷 (가능한 경우)

### 기능 제안 (Feature Requests)
새로운 기능을 제안하고 싶으다면, Issue 탭에서 'Feature Request' 템플릿을 사용해 주세요.

### Pull Request (PR) 제출
1. 이 저장소를 `Fork` 하세요.
2. 새로운 브랜치를 만드세요 (`git checkout -b feature/AmazingFeature`).
3. 변경 사항을 커밋하세요 (`git commit -m 'Add some AmazingFeature'`).
4. 브랜치에 푸시하세요 (`git push origin feature/AmazingFeature`).
5. Open a Pull Request.

## 코드 스타일 및 규칙
- **언어**: TypeScript, React, Tailwind CSS를 사용합니다.
- **포맷팅**: 프로젝트의 Prettier/ESLint 설정을 따라주세요.
- **커밋 메시지**: 명확하고 간결하게 작성해 주세요 (예: `Feat: 와인 검색 기능 추가`, `Fix: 이미지 업로드 버그 수정`).

## 개발 환경 설정
`README.md`의 '로컬 개발' 섹션을 참고하여 환경을 설정해 주세요.

## 행동 강령 (Code of Conduct)
서로 존중하고 열린 마음으로 소통해 주세요. 비방이나 부적절한 언행은 삼가 주시기 바랍니다.
